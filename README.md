Topics covered:

- Rendering shapes inside of Avalonia Canvas
- Performing a **fit-to-viewport** math on Canvas children

Adjust the size of the Canvas by changing the value in the first textbox. Aspect ratio is fixed so only the height can be changed, the width adjusts automatically.

Change the panning to move the object around.

Click Center Polygon button to adjust the panning and the scale of an object to fit perfectly into the center of the viewport.
